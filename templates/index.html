About this project

This project is a combination of multiple things.

First, it is an analysis of social media, specifically using the social media site Reddit.com. I have scraped over a million comments from Reddit.com, grouped them based on things like their political or religious affiliation, used natural language processing to conduct sentiment analyses (grouped them as positive, negative, or neutral) and looked at how different comments do in various subreddits based on these factors. You can see some example analyses in the tabs on the left of the page.

Second, I have taken these comments, the groupings I have made, and created a model to predict the comment score. You can see more in the Model Fits tab at the top of the page. I have deployed this model onn the Model page. You can reach this by clicking the Model tab on the top of the screen. This allows you to enter a comment and different features and it will predict the comment score for that comment.

Third, I have created a functional graph where you can find how well different words or phrases do on Reddit.com within the various subreddits. You can see more about that in the Main graph tab at the top of the page.
